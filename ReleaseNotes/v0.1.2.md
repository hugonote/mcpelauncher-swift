# Minecraft Bedrock Launcher 0.1.2

- Fixed first-launch preparation after Delete Data so Minecraft is warmed up again before Play starts the game.
- Fixed first-launch retry handling and warm-up logs when the runtime is not ready yet.
- Fixed manual Sparkle update checks so Check for Updates remains available when automatic launcher updates are disabled.
- Fixed Dock progress visibility during APK extraction and first-launch preparation.
- Fixed app termination when Settings is open and the main window is closed.
- Added game settings for the in-game status bar, VSync, and FPS counter.
