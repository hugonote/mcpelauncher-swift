# Minecraft Bedrock Launcher 0.1.3

- Changed Minecraft launch to run through a generated macOS app bundle so the game keeps its app icon and remains eligible for Game Mode instead of appearing as a standalone mcpelauncher-client process.
- Changed Play to skip writing Minecraft launch logs by default. Hold Option to use Play & Log when launch diagnostics are needed.
- Improved Google Play credential handling during sign-in and Minecraft launch.
- Improved offline handling so Play stays available when Minecraft and the runtime are installed.
- Minor UI improvements.
- Fixed the launcher window staying hidden after Keychain approval while first-launch runtime or Minecraft downloads continued in the background.
- Fixed first-launch downloads after Keychain Retry.
- Fixed stalled Minecraft downloads after network changes.
- Fixed incomplete macOS runtime validation.
- Fixed a possible hang when install or setup tools produced large output.
- Fixed button sizing on error screens.
